Here's an example Python function that reverses a singly-linked list:

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_list(head: ListNode) -> ListNode:
    prev = None
    current = head
    
    while current is not None:
        next_node = current.next  # save the next node
        
        current.next = prev  # point current's next to previous node (reverse link)
        
        prev = current  # move prev forward to current node
        current = next_node  # move current forward to next node
    
    return prev  # returns the new head, which is now the last element of the list
```

This function takes in a head node and iteratively reverses the direction of each link until all links are reversed. The `ListNode` class defines a generic linked list node with an integer value (`val`) and a reference to the next node (`next`). 

The function uses three pointers: `prev`, which represents the previous node; `current`, which points to the current node being processed; and `next_node`, which holds the original next node before modifying its link.

In each iteration of the loop, the code updates these pointers:
1. Saves the next node by storing it in `next_node`.
2. Reverses the direction of the link from `current` to `prev`, effectively reversing the connection between two nodes.
3. Moves `prev` forward (now pointing at `current`) and `current` forward (to the `next_node`).

The function continues this process until it reaches the end of the list, which is when `current` becomes `None`. At that point, all links have been reversed, so the new head node (`prev`, now pointing to the last element) is returned as the result.
